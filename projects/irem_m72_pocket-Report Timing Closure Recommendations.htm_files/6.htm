Description and Recommendations [hide details]

Nodes on each listed path have fanouts in other timing paths. These paths have conflicting optimization goals. This conflict results in the placer finding a globally optimal solution which is not optimal for each individual path.

The following suggestions may help resolve this issue:

  • Replicate the recommended nodes or related logic in the appropriate HDL code.
  • Enable Physical Synthesis duplication.
  • Use the Manual Logic Duplication or Maximum Fan-Out logic options to create duplicate nodes.See documentation for details on these logic options.

See the Top Recommendations section for potential solutions for each path.
